# Privacy Policy - HAR Analyzer Plugin

## Data Handling

This plugin processes HTTP Archive (HAR) files that you provide directly to it. 

### What data is processed:
- HAR file content (JSON format) containing HTTP request/response data
- URLs, headers, and response bodies from network requests
- Browser and user agent information

### How data is handled:
- All processing occurs within the Dify platform
- No data is transmitted to external servers
- No data is stored permanently after analysis is complete
- Analysis results are returned only to your Dify workflow

### What we do NOT do:
- Store your HAR files
- Share your data with third parties
- Use your data for training or analytics
- Retain any data after the tool execution completes

## User Responsibility

Users are responsible for:
- Ensuring they have permission to analyze the HAR files they upload
- Removing sensitive information (passwords, tokens, API keys) from HAR files before analysis
- Complying with their organization's data handling policies

## Contact

For questions about this plugin's data handling, please contact the plugin author.
